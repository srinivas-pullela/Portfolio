Projects

Predict Fare of Airlines Tickets using Machine Learning

Developed a machine learning model to predict airline ticket prices based on factors such as airline, route, and travel time. Utilized Python, Pandas, and Scikit-learn to preprocess data, train regression models, and visualize key pricing trends.

Predict Password Strength using Natural Language Processing

Built an NLP-based model to classify password strength as Weak, Medium, or Strong. Used Python, Scikit-learn, and TF-IDF for feature extraction and trained models using Logistic Regression and Random Forest.

Building and Deploying a Movie Recommendation System

Developed a content-based movie recommendation system using Python, Scikit-learn, and cosine similarity. Integrated the TMDb API to display movie posters and deployed the project using Streamlit for an interactive web interface.

Predict Stock Prices using Time Series Analysis

Built a time series forecasting model to predict stock prices using ARIMA and LSTM algorithms. Collected historical stock data via the Yahoo Finance API, performed data preprocessing, and visualized trends to evaluate model performance.
